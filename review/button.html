Test 1: Navigate to a button in reading mode

  • Mode: reading
  • Applies to: jaws, nvda
  • Lasted edited: Tue Dec 29 11:47:38 2020 -0500
  • Tests:
    • jaws
    • nvda
  • Relevant Specifications:
    • APG example: button/button.html

JAWS

Tester Instructions

  1. Verify the Virtual Cursor is active by pressing Alt+Delete. If it is not, turn on the Virtual Cursor by pressing Insert+Z.
  2. Navigate to the first button. using the following commands:
    • B / Shift+B
    • F / Shift+F
    • Tab / Shift+Tab
    • Up Arrow / Down Arrow
    • Left Arrow / Right Arrow (with Smart Navigation on)

Assertions

PriorityAssertion
requiredRole 'button' is conveyed
requiredName 'Print' is conveyed

NVDA

Tester Instructions

  1. Insure NVDA is in browse mode by pressing Escape. Note: This command has no effect if NVDA is already in browse mode.
  2. Navigate to the first button. using the following commands:
    • B / Shift+B
    • F / Shift+F
    • Tab / Shift+Tab
    • Up Arrow / Down Arrow

Assertions

PriorityAssertion
requiredRole 'button' is conveyed
requiredName 'Print' is conveyed
